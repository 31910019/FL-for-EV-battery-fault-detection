This code is our personalized federated learning approach for EV battery fault detection by charging data.

The data can obtained from "https://data.mendeley.com/datasets/c7gg94tmvz/2" or newer version. 

The code file "PEF_EV_ad.py" is the main flie of our method, while other files focus on comparison methods, basic algorithm components and figure drawing.
